MoCafe (Monte Carlo radiative transfer)

MoCafe is a 3D continuum radiative transfer (RT) code based on the Monte Carlo algorithm. The code is written in modern fortran and runs in parallel using MPI. MPI-3 is used to use the shared-memory in each host (node).

A Markov Chain Monte Carlo code to fit the optical/FUV observational images of spiral galaxies was developed and is being tested.
